2. Modelo Semántico

Contenido

Al iniciar el estudio de los modelos es apropiado definir a que se refiere la semántica por si existe alguna mala interpretación sobre el término. Estudio del significado de los signos lingüísticos y de sus combinaciones, desde un punto de vista sincrónico o diacrónico(Desarrollo o sucesión de hechos a través del tiempo.) (RAE, 2021) Como ya se ha mencionado la lógica de negocio es la materia prima principal para el desarrollo de una base de datos.

En este tema es inevitable hacer referencia a la lógica y a la parte conceptual del universo del negocio, elementos o entidades dentro de este, es lo que permite al diseñador de Base de datos realizar una virtualización correcta del mundo real de información.

El modelo de datos semánticos es un método de estructuración de datos para representarlos de una manera lógica específica. Es un modelo de datos conceptual que incluye información semántica que agrega un significado básico a los datos y las relaciones que se encuentran entre ellos. Este enfoque para el modelado de datos y la organización de datos permite el fácil desarrollo de programas de aplicación y también el fácil mantenimiento de la consistencia de los datos cuando se actualizan los datos, además al conocer el modelo semántico se puede preparar una estructura para adaptarse a cambios futuros.

Básicamente, el modelo semántico es una abstracción que define como los símbolos almacenados se relacionan con el mundo real. Los datos de instancia o referenciados de los modelos de datos semánticos incluyen explícitamente los tipos de relaciones entre los
diversos elementos de datos.

El modelo funciona para interpretar el significado de los hechos, típicamente estandarizan tales tipos de relaciones, ya que pueden ser integradas cuando usan los mismos tipos de relación.
